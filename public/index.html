🔐 Setup API Credentials

Untuk menggunakan app ini, kamu perlu:
1. Buat app di baruSpotify Dashboard
2. Set Redirect URI ke: https://spotify.zebua.site/spotify/
3. Copy Client ID ke form di bawah
4. Atau kalo mau gampang, kasih tau gw email lu aja 😹 (memang dari api spotify nya begitu)(api geratis wak 😹)
